本目录汇总了所有个人课程项目与个人作品的**完整源代码**，按课程分类整理。

```
projects-source/

│   ├── 实验2-虚拟内存管理/      # Python 虚拟内存页面置换算法
│   ├── 实验3-设备管理/          # Python 磁盘调度算法模拟

│   ├── 作业五-Shell成绩管理/    # Bash 脚本实现的成绩管理系统
│   └── 作业六-Flask成绩管理/    # Flask Web 成绩管理系统
├── 大数据与数据思维/            # 《大数据与数据思维》课程项目
│   ├── NarrativeAnalysis/       # 大规模电影叙事分析（Hadoop + HBase + Flask）
```
