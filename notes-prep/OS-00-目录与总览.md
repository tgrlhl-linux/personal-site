# 《操作系统》复习笔记 · 目录

## 总览

本系列笔记覆盖操作系统课程全部核心章节，按照由浅入深的顺序组织。操作系统的学习需要建立"分层抽象"的思维模式，从底层的硬件管理到上层的用户接口，每一层都在为上层提供更便捷的服务。

## 章节速览

| 章节 | 核心主题 | 重点难点 |
|------|---------|---------|
| **OS-01 引论** | 操作系统类型、中断、CPU状态 | 中断与异常的区别、状态转换 |
| **OS-02 进程管理** | 进程映像、三态转换、通信、线程 | PCB作用、通信机制选择 |
| **OS-03 CPU调度** | 调度算法、甘特图、性能计算 | 各算法的优缺点和适用场景 |
| **OS-04 同步与死锁** | PV操作、信号量、银行家算法 | 生产者-消费者、死锁避免 |
| **OS-05 内存管理** | 分区分配、分页、分段、TLB | 地址变换过程 |
| **OS-06 虚拟内存** | 页面置换、工作集、颠簸 | FIFO/LRU/OPT置换过程 |
| **OS-07 I/O系统** | I/O控制方式、缓冲、SPOOLing | DMA原理、磁盘调度 |
| **OS-08 文件系统** | 目录结构、共享、保护 | 硬链接与软链接 |
| **OS-09 磁盘管理** | 外存分配、FAT、RAID | 混合索引访盘次数计算 |

## 复习策略

1. 操作系统涉及大量概念对比（如分页vs分段、进程vs线程），建议采用"对比记忆法"
2. 计算类题目（调度算法、页面置换、银行家算法）需要动手画图，只看不练容易算错
3. 近三年真题中，PV操作与银行家算法交替出现在大题中，建议两手准备
4. 概念题主要集中在前三章和后两章，中段章节偏计算

## 符号约定

- KB = 千字节（2^10 字节）
- PCB = 进程控制块（Process Control Block）
- TLB = 快表（Translation Lookaside Buffer）
- SPOOLing = 假脱机操作（Simultaneous Peripheral Operation On-Line）
